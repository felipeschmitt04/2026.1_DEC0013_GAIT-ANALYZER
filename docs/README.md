# Documentação

Este diretório separa a documentação por área do projeto.

```text
docs/
├── backend/
│   ├── pt-br/
│   └── en/
└── frontend/
```

## Backend

`docs/backend/` contém a documentação desta entrega: API FastAPI, Docker CPU,
modo mock, worker DGX, contrato com o frontend, segurança, deploy e testes.

O índice principal fica em:

```text
docs/backend/pt-br/00-indice.md
```

## Frontend

`docs/frontend/` está reservado para a documentação do frontend, mantida pelo
responsável dessa parte do projeto.
